# P1_AnthonyMungle

Created a mock grocery store where users can buy varying products from multiple
stores.

* c#
* HTML
* SQL
* Visual Code

## Features
List of features ready
* Viewable Inventory from each store
* Multiple stores
* User Cart System
* Pictures of Inventory Items
